# Titanic
# Titanic Survival Prediction

## Overview

This project predicts whether a passenger survived the Titanic disaster using machine learning.

The dataset comes from the famous Kaggle Titanic competition.

## Dataset Features

* Passenger Class (Pclass)
* Sex
* Age
* Number of Siblings/Spouses (SibSp)
* Number of Parents/Children (Parch)
* Fare
* Embarked Port

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn
* Jupyter Notebook

## Steps Performed

1. Data Cleaning
2. Handling Missing Values
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Training
6. Prediction Generation

## Output

The trained model predicts whether a passenger **survived or not survived**.

## Files

* `titanic.ipynb` – Notebook with full analysis
* `titanic.py` – Python implementation
* `submission.csv` – Model predictions
